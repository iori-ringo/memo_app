# Memo of Magic App Walkthrough

前田裕二氏の著書『メモの魔力』にインスパイアされた、「ファクト→抽象化→転用」の実践用メモアプリを作成しました。

## 機能概要

1.  **3ステップ入力フォーム**:
    *   **ファクト (Fact)**: 客観的な事実を記録
    *   **抽象化 (Abstraction)**: 気づきや法則を抽出
    *   **転用 (Diversion)**: 具体的なアクションへ落とし込み
    *   直感的な矢印デザインとアニメーションで思考の流れをサポートします。

2.  **モダンなUIデザイン**:
    *   Next.js 15 + Tailwind CSS v4 + Shadcn/ui を採用
    *   美しく洗練されたカードデザイン
    *   ダークモード対応（システム設定に連動）

3.  **データ保存**:
    *   ブラウザのローカルストレージに自動保存されるため、リロードしてもデータが消えません。

## 実行方法

以下のコマンドで開発サーバーを起動してください。

```bash
npm run dev
```

ブラウザで `http://localhost:3000` にアクセスするとアプリが使用できます。

## ファイル構成

*   `app/page.tsx`: メインページ（ロジックとレイアウト）
*   `components/memo-form.tsx`: メモ入力フォームコンポーネント
*   `components/memo-card.tsx`: メモ表示用カードコンポーネント
*   `types/memo.ts`: メモの型定義
*   `app/globals.css`: Tailwind v4 設定とテーマカラー定義
